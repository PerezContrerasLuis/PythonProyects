Ejemplo basico de proyecto web con Django en python 

1.- creacion de proyecto en Debian 10 
2.- instalacion de Django : sudo apt install python3-pip
3.- pip --version 
4.- pip3 install Django
5.- django-admin --version 
6.- sudo apt-get install python3-django

7.- mkdir pyProject 
8.- cd pyProject
9.- python3 django-admin.py startproject MyFrstWeb

-- run local ser 
10 .- cd MyFrstWeb
11 .- python3 manage.py runserver
